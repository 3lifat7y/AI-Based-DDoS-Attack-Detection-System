AI-Based DDoS Detection

This project focuses on using Artificial Intelligence (AI) to detect Distributed Denial of Service (DDoS) attacks in real time. DDoS attacks overwhelm servers or networks with massive traffic, causing services to slow down or crash. By leveraging AI and machine learning algorithms, the system can automatically analyze network traffic patterns, identify abnormal behavior, and respond quickly to prevent service disruption.

Importance:

  • Enhances network security by detecting attacks faster than traditional methods.
  • Reduces downtime and service disruption for websites and applications.
  • Helps organizations protect sensitive data and maintain trust with users.
  • Provides an intelligent, adaptive defense that evolves with new attack patterns.

Information About DDoS Attacks

DrDoS_DNS

DNS-based Distributed Reflection DDoS. Attackers exploit open DNS resolvers to flood a target with amplified traffic.

DrDoS_LDAP

LDAP-based Distributed Reflection DDoS. Uses LDAP servers to reflect and amplify attack traffic.

DrDoS_MSSQL

Microsoft SQL-based Distributed Reflection DDoS. Leverages exposed MSSQL servers to generate large response traffic.

DrDoS_NTP

NTP-based Distributed Reflection DDoS. Exploits Network Time Protocol servers using the "monlist" command.

DrDoS_NetBIOS

NetBIOS-based Distributed Reflection DDoS. Abuses NetBIOS services in Windows systems.

DrDoS_SNMP

SNMP-based Distributed Reflection DDoS. Takes advantage of SNMP devices.

DrDoS_SSDP

SSDP-based Distributed Reflection DDoS. Uses IoT devices for amplification.

DrDoS_UDP

UDP-based Distributed Reflection DDoS. General UDP reflection attack.

LDAP

Direct LDAP flood attack. High volumes of LDAP requests.

MSSQL

Direct MSSQL flood attack. Excessive SQL requests.

NetBIOS

Direct NetBIOS flood attack. Overloads Windows services.

Portmap

Port mapping-based flood attack. Exploits RPC service.

Syn

SYN flood attack. Incomplete TCP handshakes.

TFTP

Trivial File Transfer Protocol flood attack.

UDP

UDP flood attack. Overwhelms with UDP packets.

Common DDoS Attack Types

DrDoS_DNS

DNS-based reflection attack using open DNS resolvers to amplify traffic.

DrDoS_LDAP

LDAP reflection/amplification using exposed LDAP servers.

DrDoS_MSSQL

MSSQL-based reflection attack leveraging poorly configured SQL servers.

DrDoS_NTP

NTP amplification using the "monlist" command to generate large responses.

DrDoS_NetBIOS

NetBIOS-based reflection or direct floods targeting Windows services.

DrDoS_SNMP

SNMP devices abused for reflection and amplification.

DrDoS_SSDP

SSDP/UPnP-based amplification commonly seen with IoT devices.

DrDoS_UDP

General UDP reflection attacks using various services for amplification.

LDAP

Direct LDAP flood: high volume of LDAP requests to overwhelm servers.

MSSQL

Direct MSSQL flood: excessive SQL queries causing resource exhaustion.

NetBIOS

Floods targeting NetBIOS services on Windows hosts.

Portmap

Portmap/RPC-based floods exploiting open RPC services.

Syn

SYN flood: incomplete TCP handshakes that exhaust connection tables.

TFTP

TFTP floods using trivial file transfer protocol servers for high traffic.

UDP

UDP flood attack: overwhelming targets with UDP packets.
